# COL778-Assignments

Repository containing coursework and assignment code for the COL778 course.

This repo gathers the code, notebooks, data and tools used for Assignments A1–A4.
 
## Repository structure

- **A1:** Notebooks and assets for Assignment 1 — see [A1/A1_s26.ipynb](A1/A1_s26.ipynb) and the `A1/assets` folder for URDFs and models.
- **A2:** Reinforcement-learning experiments (Ant / Hopper). Key scripts: [A2/ant-hopper/train_agent.py](A2/ant-hopper/train_agent.py), [A2/ant-hopper/test-model.py](A2/ant-hopper/test-model.py). Training data and saved models are under `A2/data`.
- **A3:** Task planning (PDDL & Fast-Downward). Main files: [A3/plan_executor.py](A3/plan_executor.py), [A3/plan.sh](A3/plan.sh), and [A3/requirements.txt](A3/requirements.txt). The Fast-Downward planner is included under `A3/downward/`.
- **A4:** Integrated control, perception and planning modules — see [A4/control.py](A4/control.py), [A4/perception.py](A4/perception.py), [A4/planning.py](A4/planning.py) and the assignment report [A4/report.pdf](A4/report.pdf).
